|--------|------|:----:|:------:|:--------:|:--------:|:--------:|---------|---------|

| **NH투자증권 QV OpenAPI** | 연결 모듈 | O | O | O | X | O | Windows | QV Auto 트레이드 내장(분할매수·매도). 나무MTS 연동 |
| **삼성증권 API** | 기관 전용 | X | O | O | O | O | 미공개 | **기관투자자 전용, 유료**. 개인 투자자 미개방 |
| **미래에셋증권 API** | MTS 전용 | X | O | O | O | X | 미공개 | 개인용 공개 자동매매 API 없음. m.Stock 앱 기반 거래 |
| **카카오페이증권** | REST(제한적) | X | X | O | X | X | 플랫폼 무관 | 개인 자동매매용 공개 API 미제공. 임베디드 금융 목적 |

[FINDING] 한국 증권사 중 개인 개발자가 자동매매에 활용 가능한 API는 **키움증권, 한국투자증권, LS증권, 대신증권, NH투자증권** 5개사다.
[STAT:n] n=8 증권사 조사
[STAT:effect_size] 5/8 (62.5%)가 개인 무료 자동매매 지원

### 국내 증권사 API 선택 기준

```
국내 주식만 + 입문자     → 대신증권 CYBOS Plus (동기방식, 개발 편의성 최고)
국내+해외 + 플랫폼 자유  → 한국투자증권 KIS Developers (REST, AI 연동, 크로스플랫폼)
국내 주식 + 저비용       → LS증권 xingAPI (수수료 저렴)
국내 주식 + 대중성       → 키움증권 OpenAPI+ (커뮤니티 자료 가장 풍부)
```

## 2. 국내 시장 데이터 소스 (Korean Market Data)

| 소스 | 무료 | 실시간 | 용도 | 제한사항 |
|------|:----:|:------:|------|---------|
| **KRX OpenAPI** (openapi.krx.co.kr) | O | X (EOD) | 주식·지수·채권·파생 공식 데이터 | 일 10,000회. 2010년 이후. **비상업 이용만** |
| **KRX Data Marketplace** (data.krx.co.kr) | 일부 | X | EOD, 지수, 특수 데이터셋 구매 | 유료 데이터셋 별도 구매 |
| **DART OpenDART** (opendart.fss.or.kr) | O | X | 전자공시(사업·분기보고서, 재무제표 JSON) | 일 10,000회. 개발자 등록 필요 |
| **PyKRX** (pip install pykrx) | O | X | KRX/네이버 스크래핑 래퍼 | **비공식**. 서비스 변경 시 중단 위험 |

## 8. 주요 제한사항

[LIMITATION] **Windows 종속성**: 키움·CYBOS·xingAPI COM 기반 API는 Windows 전용. Linux/macOS 환경에서는 한국투자증권 KIS만 사용 가능.

[LIMITATION] **Yahoo Finance 비공식 의존**: yfinance는 공식 API가 아님. 서비스 구조 변경 시 데이터 파이프라인 중단 가능. 장기 프로덕션에 부적합.

[LIMITATION] **TD Ameritrade API 종료**: 2024년 5월 종료됨. 기존 사용자는 Schwab Trader API 또는 Alpaca로 마이그레이션 필요.

[LIMITATION] **KRX/DART 비상업 조건**: 공식 거래소 데이터는 비상업적 이용만 허용. 상업 서비스에는 유료 데이터 계약 필요.

[LIMITATION] **실시간 데이터 한계**: 무료 API는 대부분 15~20분 지연 데이터. 실시간 데이터는 증권사 계좌 연동 또는 유료 구독 필요.

[LIMITATION] **레이트 리밋**: Alpha Vantage 무료 한도(25req/day)는 프로덕션 시스템에 매우 부족. Polygon.io 무료(5req/min)도 마찬가지. 운용 전 유료 플랜 검토 필수.

[LIMITATION] 본 조사는 2026년 3월 기준 웹 검색 결과 및 공식 문서 기반. API 정책·가격은 수시로 변경될 수 있으므로 각 서비스 공식 사이트에서 최신 정보 확인 권장.
